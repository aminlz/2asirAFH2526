8 Capturas del Proceso Completo LLM Fútbol

Análisis detallado de cada etapa desde conversión GGUF hasta modelo funcionando

✅ PROCESO COMPLETADO · Fine-tuning Exitoso

CAPTURA 1

Conversión del modelo a formato GGUF

1
En esta imagen se observa la terminal ejecutando el proceso de conversión del modelo a formato GGUF, que es el formato utilizado para ejecutar modelos de lenguaje de forma local. El sistema inicia la cuantización del modelo y comienza a escribir los tensores necesarios dentro del archivo modelo_futbol_gguf.

Durante el proceso aparece el progreso hasta llegar al 100%, indicando que todos los datos del modelo han sido exportados correctamente. Finalmente, el mensaje confirma que el modelo se ha exportado con éxito, lo que significa que ya está preparado para ser usado en aplicaciones locales de inteligencia artificial.

CAPTURA 2

Conjunto de datos JSON para entrenamiento

2
En esta imagen aparece el conjunto de datos utilizado para entrenar el modelo. Se muestra un formato tipo JSON donde cada ejemplo contiene instrucciones, entradas y respuestas relacionadas con el fútbol.

Estos datos sirven para enseñar al modelo cómo debe responder, incluyendo preguntas sobre historia, reglas, jugadores y competiciones. Gracias a este dataset, el modelo aprende el estilo de respuesta y el conocimiento específico que luego aplicará durante las conversaciones.

CAPTURA 3

Preparación Unsloth + HuggingFace

3
Aquí se muestra el inicio del proceso de preparación del entrenamiento mediante la herramienta Unsloth junto con HuggingFace. El sistema revisa la caché local para comprobar si los archivos necesarios del modelo ya existen.

Al detectar que algunos archivos faltan, comienza automáticamente la descarga de los pesos del modelo en formato safetensors. Este paso es necesario para poder continuar con el entrenamiento personalizado.

CAPTURA 4

Inicio del entrenamiento y métricas

4
En esta captura comienza el entrenamiento del modelo. Se muestran diferentes parámetros técnicos como el número de ejemplos, el tamaño de los lotes (batch size), las épocas de entrenamiento y la tasa de aprendizaje.

También aparece el valor de "loss", que mide el error del modelo. A medida que avanza el entrenamiento, este valor va disminuyendo, lo que indica que el modelo está aprendiendo correctamente a generar mejores respuestas.

CAPTURA 5

Finalización del fine-tuning

5
En esta imagen se observa la parte final del entrenamiento. Los registros continúan mostrando cómo el valor de pérdida sigue bajando hasta completar todas las épocas programadas.

Al finalizar, el sistema guarda automáticamente el modelo entrenado y muestra un mensaje confirmando que el modelo de fútbol ha sido almacenado correctamente. Esto significa que el fine-tuning ha terminado con éxito.

CAPTURA 6

Modelos instalados en el sistema

6
Esta captura muestra la interfaz donde aparecen los modelos instalados en el sistema. Se puede ver el modelo personalizado junto con otros modelos disponibles, incluyendo información como el tamaño, tipo de cuantización y fecha de modificación.

La presencia del modelo en esta lista confirma que ha sido reconocido correctamente por el programa y está listo para utilizarse.

CAPTURA 7

Configuración del preset "Fútbol asistente"

7
En esta imagen se visualiza la configuración de parámetros del modelo. Se ha creado un preset llamado "Fútbol asistente" y se define un System Prompt que establece el comportamiento del modelo.

El texto indica que el asistente debe actuar como experto en fútbol, explicando reglas, historia y estadísticas de forma clara y en español. Esto controla la personalidad y el estilo de respuestas del modelo.

CAPTURA 8

Modelo funcionando en chat

8
En la última captura se muestra el modelo ya funcionando dentro del chat. El usuario realiza una pregunta sobre el fuera de juego y el modelo responde con una explicación estructurada y detallada.

Esto demuestra que todo el proceso anterior —conversión, entrenamiento y configuración— ha funcionado correctamente, ya que el modelo responde siguiendo el rol de asistente experto en fútbol.
